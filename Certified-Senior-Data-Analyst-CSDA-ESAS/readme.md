📚 Kurikulum & Materi Utama

Program ini mencakup alur kerja end-to-end seorang Data Analyst senior, meliputi:

    Fondasi Data: Pengantar Ilmu Data serta Peluang dan Tantangannya di industri saat ini.

    Manajemen Data: Teknik Pengumpulan Data dan Pembersihan Data (Data Cleaning).

    Teknis & Alat: Bahasa Pemrograman untuk Analisis Data dan Praktik Analisis Data menggunakan Python.

    Metodologi Statistik: Teknik Statistik untuk Analisis Data, Proses Analisis, dan Pemodelan Data.

    Output Bisnis: Interpretasi Hasil dan Penyajian Data (Visualisasi).
