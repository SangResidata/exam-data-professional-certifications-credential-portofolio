## 📚 Materi Pelatihan
Pelatihan ini mencakup siklus hidup analisis data end-to-end:
1.  **Pengantar Ilmu Data:** Peluang dan tantangan di industri.
2.  **Data Ingestion:** Teknik pengumpulan data.
3.  **Data Preprocessing:** Pembersihan data (Data Cleaning).
4.  **Tools & Tech:** Bahasa pemrograman untuk analisis data.
5.  **Statistik:** Teknik statistik untuk analisis data.
6.  **Workflow:** Proses analisis dan pemodelan data.
7.  **Output:** Interpretasi hasil dan penyajian data (Visualisasi).
8.  **Hands-on:** Praktik analisis data menggunakan **Python**.
